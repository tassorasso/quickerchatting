# quickerchatting
uses p2p to connect

# how to connect to a party
1. Enter a display name at the start screen
2. Enter the Password (if it's a private party)
3. Enter the Host IP at the start screen
4. Enter the Port
5. Press "Join Party"

# how to make a private Party
1. Enter a display name at the start screen
2. Enter a Password
3. Enter the Host IP at the start screen
4. Enter the Port
5. Press "Host Party"

# how to make a public Party
1. Enter a display name at the start screen
3. Enter the Host IP at the start screen
4. Enter the Port
2. Enable the "Public Server" Checkbox
5. Press "Host Party"
